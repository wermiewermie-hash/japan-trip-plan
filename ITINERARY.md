## How to edit this doc

Edit anything below and send it back — I'll sync your changes into the site.

**Format for each day:**

- **Day / City / Date / Title / Location / Highlight** are the labelled fields at the top of each day.
- Each **timeline item** is a bullet. The pieces of a bullet:
  - `[time]` — optional. Leave off if the item has no set time.
  - The **emoji** comes next (keep or swap it — it's the little icon shown).
  - Then the **text** — the main line.
  - `_(italic in parentheses)_` — an optional **note/subtitle** shown under the text.
  - `**TODO**` at the end — flags the item as still-to-decide (shows a yellow "to do" tag). Delete it once it's settled.
  - `→ [Label](url)` — optional **links**. Add as many as you like on their own indented sub-bullets.

**To add an item:** copy a bullet's shape. **To remove one:** delete the bullet. **To reorder:** move bullets around. **To add a whole day:** copy a `### Day N` block.

Don't worry about getting the punctuation perfect — I'll clean it up when I import.
